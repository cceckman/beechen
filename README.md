# Beechen

RC Creative Coding, 2024-05-22

This week's prompt is from [Keats' "Ode to a Nightingale"](https://www.poetryfoundation.org/poems/44479/ode-to-a-nightingale):

> Of [beechen](https://en.wiktionary.org/wiki/beechen) green, and shadows numberless,
>   Singest of summer in full-throated ease.
